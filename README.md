# Bank Marketing Prediction using Decision Tree

## Project Overview

This project develops a **Decision Tree Classification model** to predict whether a bank customer is likely to subscribe to a term deposit based on demographic details, financial information, and previous marketing interactions.

The project follows a complete machine learning pipeline, including data preprocessing, exploratory data analysis (EDA), feature preparation, model development, performance evaluation, and result visualization.

This work was completed as **Task 03** during the **Prodigy InfoTech Data Science Internship**.

---

# Objective

The primary objective of this project is to build a classification model capable of predicting customer responses to a bank marketing campaign. Such predictions can help financial institutions identify potential customers and improve the effectiveness of future marketing strategies.

---

# Dataset Information

**Dataset:** Bank Marketing Dataset

**Source:** UCI Machine Learning Repository

The dataset contains customer demographic, financial, and campaign-related attributes, including:

* Age
* Occupation
* Marital Status
* Education Level
* Account Balance
* Housing Loan Status
* Personal Loan Status
* Contact Method
* Campaign Details
* Previous Marketing Outcomes

**Target Variable**

**y**

* **Yes** – Customer subscribed to the term deposit
* **No** – Customer did not subscribe
